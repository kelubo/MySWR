# HF双定向耦合器设计

## 概述

这是一个用于SWR驻波表项目的双定向耦合器设计，工作频率范围为1-30 MHz（HF频段）。该设计采用Kicad格式，包含完整的原理图和PCB布局。

## 技术规格

- **工作频率**: 1-30 MHz
- **耦合度**: 20 dB
- **方向性**: >25 dB
- **插入损耗**: <0.5 dB
- **阻抗**: 50 Ω
- **PCB尺寸**: 100 x 100 mm
- **PCB厚度**: 1.6 mm
- **PCB层数**: 2层

## 接口定义

- **J1**: 输入端口 (SMA 边缘安装)
- **J2**: 输出端口 (SMA 边缘安装)
- **J3**: 正向耦合端口 (SMA 边缘安装)
- **J4**: 反向耦合端口 (SMA 边缘安装)
- **P1-P3**: 安装孔 (2.2 mm 直径)
- **R1-R2**: 50 Ω 负载电阻

## 文件说明

- `directional_coupler_hf.kicad_pro`: Kicad项目配置文件
- `directional_coupler_hf.kicad_sch`: 原理图文件
- `directional_coupler_hf.kicad_pcb`: PCB布局文件
- `directional_coupler_hf.kicad_prl`: Kicad项目布局文件
- `README.md`: 设计文档

## 使用方法

1. 使用Kicad 7.0或更高版本打开项目文件
2. 查看和修改原理图(如果需要)
3. 检查PCB布局并进行必要的调整
4. 生成Gerber文件用于PCB制造
5. 焊接所有组件

## 设计特点

- 采用微带线耦合器设计
- 优化的50 Ω阻抗匹配
- 良好的方向性，减少信号串扰
- 边缘安装SMA连接器，方便与其他设备连接
- 合理的安装孔布局，便于固定

## 应用领域

- SWR驻波比测量
- 功率监测
- 天线系统调试
- 射频信号分析

## 注意事项

- 确保PCB制造使用FR-4材料
- 建议使用高质量的SMA连接器
- 焊接时注意避免过热损坏组件
- 调试时使用矢量网络分析仪(VNA)验证性能
