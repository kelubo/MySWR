# VHF/UHF双定向耦合器设计

## 概述

这是一个用于SWR驻波表项目的VHF/UHF频段双定向耦合器设计，工作频率范围覆盖30-500 MHz的业余无线电频段。该设计采用Kicad格式，包含完整的原理图和PCB布局，与现有的1-30 MHz耦合器设计完全独立。

## 技术规格

- **工作频率**: 30-500 MHz
- **耦合度**: 20 dB
- **方向性**: >20 dB
- **插入损耗**: <0.5 dB
- **阻抗**: 50 Ω
- **PCB尺寸**: 60 x 60 mm
- **PCB厚度**: 1.6 mm
- **PCB层数**: 2层
- **板材**: FR-4
- **铜厚**: 35 μm

## 业余无线电频段覆盖

该设计覆盖以下业余无线电频段：

| 频段 | 频率范围 |
|------|----------|
| 6米  | 50-54 MHz |
| 2米  | 144-148 MHz |
| 1.25米 | 222-225 MHz |
| 70厘米 | 420-450 MHz |

## 接口定义

- **J1**: 输入端口 (SMA 边缘安装)
- **J2**: 输出端口 (SMA 边缘安装)
- **J3**: 正向耦合端口 (SMA 边缘安装)
- **J4**: 反向耦合端口 (SMA 边缘安装)
- **R1-R2**: 50 Ω 负载电阻 (0805封装)

## 文件说明

- `directional_coupler_vhf_uhf.kicad_pro`: Kicad项目配置文件
- `directional_coupler_vhf_uhf.kicad_sch`: 原理图文件
- `directional_coupler_vhf_uhf.kicad_pcb`: PCB布局文件
- `README.md`: 设计文档

## 使用方法

1. 使用Kicad 7.0或更高版本打开项目文件
2. 查看和修改原理图(如果需要)
3. 检查PCB布局并进行必要的调整
4. 生成Gerber文件用于PCB制造
5. 焊接所有组件
6. 使用矢量网络分析仪(VNA)验证性能

## 设计特点

- 采用微带线耦合器设计，优化用于VHF/UHF频段
- 良好的50 Ω阻抗匹配
- 边缘安装SMA连接器，方便与其他设备连接
- 双层PCB设计，底层为完整接地平面
- 紧凑的PCB尺寸，便于集成到小型设备中
- 独立的设计，与1-30 MHz耦合器完全兼容

## 应用领域

- VHF/UHF业余无线电SWR驻波比测量
- 功率监测
- 天线系统调试
- 射频信号分析

## 注意事项

- 确保PCB制造使用高质量FR-4材料
- 建议使用高质量的SMA连接器
- 焊接时注意避免过热损坏组件
- 调试时使用矢量网络分析仪(VNA)验证性能
- 建议在测试时使用低功率信号源，避免损坏耦合器
